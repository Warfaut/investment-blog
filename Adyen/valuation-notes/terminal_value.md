# Terminal Value — Gordon Growth Method (Base Case)

The terminal value is estimated using the Gordon Growth Model, assuming
a stable long-term growth rate beyond the explicit forecast period.

### Key Assumptions

- Terminal growth rate (g): 2.5%
- WACC: 8.8%
- Terminal year FCF (2028E): €1,105m

### Terminal Value Calculation

TV = FCF_2028 × (1 + g) / (WACC − g)

TV = 1,105 × 1.025 / (0.088 − 0.025) ≈ €18,000m
